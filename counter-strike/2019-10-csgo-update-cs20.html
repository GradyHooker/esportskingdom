New Case and New Cache in Counter-Strike update

Grady Hooker (@GradyHooker)

2019.10.18

Valve just released a new update for Counter-Strike: Global Offensive that introduces the new version of Cache into the game as well as a new theme Weapon Case and Sticker Capsule. We take a look at what is inside the cases as the other miscellaneous changes in the update.

New Cache Version

FMPONE and Volcano's new version of Cache, first seen at ESL One New York in the showmatch, is now playable on official servers. It is available in Casual, Deathmatch, and Scrimmage.

Miscellaneous

Alongside the new case and new version of Cache being added to the game, there was some smaller miscellaneous changes.

  • Added a game launch setting -lowmemworkshop to allow updating very large map files on Steam Workshop
  • Added a runtime command fov_tv_debug which allows changing field view in roaming mode in GOTV or during demo playback
  • Enabled area portals performance optimizations
