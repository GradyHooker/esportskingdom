CSGO introduces a new case, 3 new weapon collections, and new stickers and graffiti

Grady Hooker (@GradyHooker)

2019.11.19

In the second part of today's Counter-Strike update, Valve added a bunch of new community content to the game. We have three brand new weapon collections (Norse Collection, St. Marc Collection, Canals Collection), some new stickers, and some new graffiti.

Shattered Web Case

First up is the brand new case to be used alongside the new Shattered Web Operation. It includes four new knives; the Paracord handled field knife, the Survival, the Nomad, and the Skeleton knife.

The Norse Collection

Secondly we have the Norse Collection which features designs by The Honey Badger and contains 18 different designs.

The St. Marc Collection

Thirdly is the St. Marc Collection which contains 18 unique weapon skins created by Chemical Alia.

The Canals Collection

Lastly for the weapons collections is everyones favorite map, Canals, and the weapon collection contains designs by Teo~.

Stickers and Graffiti

With all of these skins, Valve has also released a set of Shattered Web inspired skins designed by community artist daniDem.
